# cn-index-notes

本仓库用于归档和发布多个独立 HTML 页面，旨在以结构化的方式保存与整理各类索引性质的内容。

## 项目简介

`cn-index-notes` 是一个静态页面归档仓库，主要存放独立的 HTML 文件。这些文件面向通用索引场景，不针对特定域名或网站。仓库的定位是提供一个干净、可浏览的页面集合，便于长期维护与访问。

## 目录说明

```
.
├── index.html          # 仓库入口页，列出所有归档页面
├── pages/              # 存放所有独立的 HTML 文件
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── README.md           # 本文件
```

- `index.html`：仓库的主页，用于导航到各个归档页面。
- `pages/`：核心目录，每个 HTML 文件为一个独立归档单元。

## 页面归档说明

- 所有页面均为纯 HTML 格式，不依赖外部资源（如 CSS 库、JavaScript 框架）。
- 页面内容聚焦于索引与整理，不含商业推广或诱导点击元素。
- 文件名采用简洁的命名规则，便于识别与排序。

## 维护说明

- 欢迎通过 Pull Request 提交新的归档页面，或对现有页面进行修正。
- 提交前请确保页面内容符合通用归档目的，不涉及具体域名或商业信息。
- 仓库维护者会定期审核提交内容，保持整体风格一致。
